Manual Setup
 - Set search engine to DuckDuckGo
 - Add a searx instance
 - Move new tab button to top of window instead of on tab bar
 - Lighten the strictness of NoScript
 - Make uBlock Origin more strict
 - Disable darkreader by default (Alt+Shift+D)
 - Enable BitWarden and uBlock Origin in private tabs
 - Pack and load hh3 as an extension
